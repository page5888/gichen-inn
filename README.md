# 吉城客棧 - 官方宣傳網頁

花蓮吉安鄉隱藏版美食餐廳的高品質單頁式宣傳網站。

## 特色

- 🎨 **UI-UX-Pro-Max 設計系統**：嚴謹的色彩與字型規範
- ♿ **WCAG AA 無障礙設計**：完整的 aria-label 與語意化 HTML5
- 📱 **響應式設計**：完美適配各種裝置
- 🔍 **SEO/AEO 優化**：LocalBusiness JSON-LD 結構化資料
- ✨ **Impeccable 美學**：大膽排版、精緻微動效

## 內容

- **Hero Section**：首頁主視覺，強烈視覺張力
- **About Us**：品牌故事，圖文交錯排版
- **Services**：非對稱 Bento Box 菜色展示
- **FAQ**：互動手風琴常見問題
- **Contact**：聯絡資訊與 Google Maps 嵌入
